API de Gestão de Contactos

1 user pode ter mais de um contacto + 1 contacto só tem um user.
Só quem esta autenticado pode add contacto, buscar usando filtro, atualizar, e deletar.

get all - de todos os contactos e os seus respectivos nomes.

os contactos tevem ter no maximo só 16.

usar uma API externa para verificar o prefixo, para saber se o numero é de que pais. e salvar no banco o nome do pais.

se for apagar um User deve apagar todos os contactos a ele vinculados.

Na tabela de contacto deve ter o campo:
prefixo = ex: +244 >= 4
pais = ex: Angola
numero = 923456780 >= 11

Na tabela User deve ter o campo:
name = ex: Mingous

API DE Gestão de Contactos

get - listar contactos
get - filtrar contactos de nome e numero
post - create contacto
put - atualizar contacto
delete - remove contaacto

tecnologias:
TypeScript, postgress, knex, zod, docker, tests com jest, JWT, opcional: monitorar aplicação de node + ts
